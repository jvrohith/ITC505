Create Your Mad Lib

Addendum

This Mad Lib was created as part of the ITC505 Lab 7 project. The goal of the assignment is to create a simple web-based Mad Lib generator using ExpressJS and a POST request to handle user inputs.

Process:

  • The user is prompted to enter words such as nouns, verbs, adjectives, etc.
  • Upon form submission, the values entered by the user are sent to the server.
  • The server processes the inputs and generates a Mad Lib using string concatenation.
  • The result is then displayed back to the user on the same page.

Design Choices:

  • Used a simple form layout with five input fields for the Mad Lib components.
  • Applied CSS styling to improve the user experience, with a clean, minimal design that focuses on the form and its results.
  • Made use of ExpressJS for handling both GET and POST requests, which allows for dynamic data handling and response generation.

This is a simple yet effective way to introduce students to web forms, server-side processing, and string manipulation in JavaScript. It also demonstrates how to build interactive pages using ExpressJS and HTML forms.
